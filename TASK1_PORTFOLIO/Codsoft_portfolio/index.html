Projects

District Level Election Commission Website

The District Level Election Commission Website is a comprehensive platform designed to streamline election management at the district level. It features voter registration, candidate information, election schedules, and real-time results updates. The website ensures transparency and accessibility, providing detailed election-related information to the public. It also includes news broadcasting, event notifications to all logged-in users, a youth engagement section, and a complaint box. This website helps administrators such as MLAs or Aamdars stay connected with the civil populace, enhancing communication and engagement between officials and the community. Developed with a user-friendly interface, it facilitates easy navigation and engagement, enhancing the overall electoral process within the district.

Safe-Heaven : A Woment Safety Application

The Women Safety App is an Android application designed to enhance personal safety for women. Built using Firebase, it offers real-time location tracking, emergency contacts alerts, and a quick SOS button. When activated, the app sends the user's location to pre-selected contacts, ensuring immediate assistance. The app also includes emergency voices such as police sirens and gun firing, along with voice recording capabilities to capture real-time audio evidence. Firebase integration ensures seamless data management, user authentication, and reliable performance, making the app a robust tool for women's safety.

PlanPerfect: Event Planning Website

PlanPerfect is an event planning platform developed using Django, featuring secure user registration and authentication. Users can create and manage events, update their profiles, join events, and leave reviews on attended events. The platform also allows event sharing and integrates calendar functionality to track important dates. Additional features include managing the admin panel as an owner, sending reminders to join members, a feedback form, a dynamic slider, and statistical analysis with pie charts and line charts. Django is used for both the backend and database storage, streamlining event organization for efficient and effective planning.

Quiz App

The Quiz App is an interactive platform developed using Android. It features an admin panel for adding different quizzes via file upload or manual inputs. Users can register securely and solve quizzes with a timer, enhancing the challenge. Upon completion, the app displays the user's score, providing immediate feedback. This app ensures a seamless and engaging user experience, promoting effective learning and self-assessment.
